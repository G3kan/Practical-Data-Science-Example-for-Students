# Practical-Data-Science-Example-for-Students
Bu depo, veri bilimi ve veri mühendisliği öğrencileri için temel kavramları gerçek dünya senaryolarıyla birleştiren bağımsız uygulama örnekleri içerir. Her klasör, verinin ham halden alınarak anlamlı bir içgörüye dönüşme sürecindeki farklı bir aşamayı temsil eder.
