Bu proje, Python kullanarak bir CSV dosyasındaki verileri PostgreSQL veritabanına taşımayı ve Metabase üzerinden analize hazır hale getirmeyi gösterir.

Yapılan İşlemler
Bağlantı: Python ile SQL veritabanı arasında iletişim kurulur.

Kontrol: İşlem öncesi ve sonrası veritabanındaki tablo sayıları ekrana basılır.

Aktarım: Sonhali.csv dosyası okunarak veritabanındaki Satislar tablosuna yüklenir.

Çalıştırma Notu
Kod çalıştırılmadan önce veritabanı kullanıcı adı, şifre ve dosya yolunun doğruluğundan emin olunmalıdır.
